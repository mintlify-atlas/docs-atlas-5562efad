The `br-2024.py` script processes the full multi-season `campeonato-brasileiro-full.csv` dataset to surface structural patterns: which clubs dominate at home, which classical derbies are most lopsided, and how Elo ratings evolved year by year.

## Running the analysis

<Steps>
  <Step title="Install matplotlib">
    The script generates an Elo evolution chart and requires matplotlib:

    ```bash
    pip install matplotlib pandas tabulate
    ```
  </Step>
  <Step title="Run the script">
    ```bash
    python br-2024.py
    ```

    The script prints the insights tables to stdout and opens a matplotlib window showing Elo trajectories over time.
  </Step>
  <Step title="Inspect the chart">
    The chart plots one line per focus club, with the x-axis representing calendar year and the y-axis representing Elo rating. Periods of sustained dominance appear as upward trends; relegation seasons appear as sharp drops.
  </Step>
</Steps>

<Note>
All insights below are derived from `campeonato-brasileiro-full.csv`, which covers multiple Série A seasons. Conclusions reflect long-run historical patterns, not the current season.
</Note>

## Biggest Elo growth in the analyzed period

Of the 12 focus clubs, **Palmeiras** recorded the greatest net Elo gain across the full dataset:

```text
Club with greatest Elo growth: palmeiras (+172.08 points)
```

Elo growth is calculated as the difference between a club's final Elo and their opening Elo of 1500:

```python
crescimento = df_elos.iloc[-1] - df_elos.iloc[0]
maior_crescimento = crescimento.idxmax()       # 'palmeiras'
maior_crescimento_valor = crescimento.max()    # +172.08
```

A gain of +172 points over the full dataset places Palmeiras roughly 66% likely to beat a club sitting at the average 1500 rating.

## Data source

All figures above are computed from `campeonato-brasileiro-full.csv`. The file contains Série A match records with the following relevant columns:

| Column | Description |
|--------|-------------|
| `mandante` | Home club name |
| `visitante` | Away club name |
| `gols_mandante` | Goals scored by home club |
| `gols_visitante` | Goals scored by away club |
| `resultado` | Match result (`vitoria_mandante`, `empate`, `vitoria_visitante`) |

Team names are normalized to lowercase and stripped of leading/trailing whitespace before processing.
